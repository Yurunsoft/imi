# Workerman

imi v2.0 版本开始，支持运行在 Workerman 环境中。

组件引入：`composer require imiphp/imi-workerman`

## 核心特性

| 特性 | 是否支持 |
|-|-
| Http | ✔ |
| Http2 | ✔ |
| WebSocket | ✔ |
| TCP | ✔ |
| UDP | ✔ |
| MQTT |  |

## 命令

启动服务: `vendor/bin/imi-workerman workerman/start`

停止服务: `vendor/bin/imi-workerman workerman/stop`
